# 擬真網頁計算機 V1

這是一個使用 **HTML、CSS、JavaScript** 製作的簡易網頁計算機。專案不需要安裝任何套件，只要用瀏覽器開啟 `index.html` 即可使用。

## 專案特色

- 使用純前端技術開發，無需後端伺服器。
- 支援數字輸入 `0` 至 `9`。
- 支援基本加法 `+` 與減法 `-`。
- 支援等號 `=` 計算結果。
- 支援清除鍵 `C` 重置目前輸入與運算狀態。
- 具有擬真按鍵陰影與按下動畫效果。
- 使用 CSS Grid 建立整齊的按鍵排列。

## 專案結構

```text
.
├── index.html   # 頁面結構與計算機按鈕配置
├── style.css    # 計算機外觀、排版與按鈕樣式
└── script.js    # 計算邏輯、輸入處理與狀態管理
```

## 檔案說明

### `index.html`

`index.html` 是專案的主要入口檔案，負責建立計算機的 HTML 結構。內容包含：

- 計算機外層容器 `.calculator`
- 顯示計算結果的區塊 `#display`
- 數字按鈕 `0` 至 `9`
- 運算按鈕 `+`、`-`、`=`
- 清除按鈕 `C`
- 載入 `style.css` 與 `script.js`

### `style.css`

`style.css` 負責整體視覺設計與版面配置，包含：

- 使用 Flexbox 將計算機置中於畫面
- 計算機主體的背景、圓角與陰影
- 顯示螢幕的深色背景與靠右對齊文字
- 使用 CSS Grid 建立四欄按鍵區
- 不同類型按鈕的顏色區分
  - 一般數字按鈕：灰色
  - 運算符號按鈕：橘色
  - 清除按鈕：紅色
- 按鈕按下時的位移動畫

### `script.js`

`script.js` 負責計算機的互動與運算邏輯，主要包含以下狀態變數與函式：

#### 狀態變數

| 變數 | 說明 |
| --- | --- |
| `currentInput` | 目前正在輸入的數字 |
| `previousInput` | 運算前輸入的第一個數字 |
| `currentOperator` | 目前選擇的運算子，例如 `+` 或 `-` |

#### 主要函式

| 函式 | 功能 |
| --- | --- |
| `updateDisplay(value)` | 更新畫面顯示內容 |
| `appendNumber(number)` | 將使用者點擊的數字加入目前輸入 |
| `setOperator(operator)` | 設定目前運算子，並保存第一個輸入值 |
| `calculate()` | 根據目前運算子執行加法或減法 |
| `clearDisplay()` | 清空輸入、運算子與顯示畫面 |

## 使用方式

1. 將三個檔案放在同一個資料夾中：
   - `index.html`
   - `style.css`
   - `script.js`
2. 使用瀏覽器開啟 `index.html`。
3. 點擊計算機按鈕開始操作。

## 操作範例

### 加法

```text
輸入：7 + 8 =
結果：15
```

### 減法

```text
輸入：9 - 4 =
結果：5
```

### 錯誤情境

若在沒有完整算式時直接按下 `=`，畫面會顯示：

```text
錯誤
```

接著再次輸入數字時，程式會自動清除錯誤狀態。
